# Get started

### **Example of usage of components**

Once you add the package, update your .storybook/config.js to import all files ending in .stories.js.

```
import { Button } from "@components/atoms/Button"
...

<Button variant="primary">Click me!</Button>
```

### **Run and develop locally**

Clone the [project](https://github.com/Nokkvi96/quiz-app) then install and start Storybook.

`npm install storybook npm run storybook`

Note: this package is not used in Storybook's UI, but the visual design is identical.
